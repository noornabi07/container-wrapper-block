Description of the Container Block

== Description ==
Wrapper/Container Block is a Gutenberg plugin that allows you to group multiple blocks into a single container for better layout control and styling.

Large Description:
Wrapper/Container Block is a powerful Gutenberg plugin designed to enhance your WordPress content creation experience by allowing you to group multiple blocks within a single container. This plugin is ideal for users who want to create complex layouts and manage grouped elements more efficiently.
Maximize your content creation potential with the Wrapper/Container Block plugin, and enjoy a more organized, efficient, and visually appealing WordPress site.



**[Demos](https://bblockswp.com/demo/block-name)**


= Features =
- **Fully Customizable: You can create your own design as per your choice.
- ** Easy Grouping: Effortlessly wrap multiple blocks into a single container, simplifying layout management and organization.
- ** Custom Styling: Apply unique styles to the container and its contents, providing greater control over the visual appearance of your content.
- ** Responsive Design: Ensure your grouped content looks great on all devices with built-in responsive design options.
- ** User-Friendly Interface: Enjoy a simple and intuitive interface that makes the plugin easy to use for both beginners and advanced users.
- ** Enhanced Flexibility: Perfect for creating sections, columns, and other intricate layout structures within the Gutenberg editor.
- ** Reusable Blocks: Save container blocks as reusable blocks to streamline your workflow and maintain consistency across your site.
- ** Advanced Controls: Access advanced settings to fine-tune the behavior and appearance of your container blocks.


= How to use =
- First, install the Container Block plugin
- Add the Container Block from the block category called "Widgets" in Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Container/wrapper Block**
3. Click on the **Container/wrapper Block** to add the block

= Download & Upload: =
1. Download the **Container Block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`plugin-slug.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Container Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
